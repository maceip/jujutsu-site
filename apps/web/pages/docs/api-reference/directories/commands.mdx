# `commands/`

The **commands** directory houses the slash commands that users use to interact with your Discord bot.

```typescript commands/hello.ts
export const description = 'Hello world!'

export default function Hello({ interaction }: Discord.CommandProps) {
  interaction.reply(`Hello, ${interaction.user?.tag}!`)
}
```

### Props

#### `interaction`

This is the interaction object passed through Discord.js. Learn more [here](https://discord.js.org/docs/discord.js/main/class/ChatInputCommandInteraction)

#### `client`

The Discord bot that you are currently logged into. Learn more [here](https://discord.js.org/docs/discord.js/main/class/Client)
